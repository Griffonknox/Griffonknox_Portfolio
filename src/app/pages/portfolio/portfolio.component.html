SSL Certificate Auto Renewal

ASP.NET C# - WPF, Entity Framework, Win-Acme
A desktop tool that automates SSL certificate creation and renewal using Win-Acme, with full integration for Azure-hosted applications. The app provides a WPF-based GUI for managing certificate bindings and configurations, making it easy to monitor and control the lifecycle of certificates. Certificate metadata is stored and accessed via Entity Framework, and the system automatically updates and rebinds certificates upon expiration. This project streamlines secure deployment workflows for web applications hosted on Azure.
